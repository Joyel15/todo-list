# To-Do List

A simple to-do list app built with HTML, Tailwind CSS and JavaScript.

## Features
- Add tasks
- Delete tasks
- Tasks exist after page reload using localStorage

## How to run
1. Clone the repo
2. Run `npm install`
3. Open `src/index.html` in your browser
